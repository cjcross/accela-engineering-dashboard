## Getting started
* ```npm install -g express```
* ```npm install -g nodemon```
* ```npm install```
* ```npm start```

## Node Env Variables
You will need to set the following environmental variables
* ```VSTS_KEY```
* ```VSTS_URI```

i.e. ```export VSTS_KEY=your-secret-key-goes-here``` from the command line on mac osx
